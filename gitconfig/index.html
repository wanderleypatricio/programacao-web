📘 Git Cheat Sheet

Referência rápida com explicações dos principais comandos Git

⚙️ Configuração Inicial

Define seu nome e email para os commits do Git.

git config --global user.name "Seu Nome"
git config --global user.email "seu@email.com"

📁 Iniciar um Repositório

Cria um novo repositório Git local na pasta atual.

git init

🔍 Verificar o Status

Mostra arquivos modificados, adicionados ou não rastreados.

git status

➕ Adicionar Arquivos

Prepara arquivos para o próximo commit.

git add arquivo.txt       # Adiciona um arquivo específico
git add .                 # Adiciona todos os arquivos modificados

💬 Commit

Salva as mudanças no histórico do Git.

git commit -m "mensagem descritiva"

🚀 Push & Pull

Push envia commits locais para o repositório remoto. Pull traz atualizações do remoto para local.

git push origin main
git pull origin main

🌿 Branches

Trabalhe com versões paralelas do projeto.

git branch                  # Lista branches
git branch nova-feature     # Cria nova branch
git checkout nova-feature   # Muda para a nova branch
git checkout -b nova-feature # Cria e já muda para a branch
git branch -d nome-da-branch # Deleta uma branch local

🕓 Histórico

Mostra os commits feitos no projeto.

git log                      # Exibe detalhes de cada commit
git log --oneline --graph    # Visual compacta com ramificações
